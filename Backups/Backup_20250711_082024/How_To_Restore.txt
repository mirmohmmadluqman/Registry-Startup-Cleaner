HOW TO RESTORE REGISTRY ENTRIES 
================================ 
 
METHOD 1: Manual Restore 
----------------------- 
1. Press Win + R, type 'regedit', press Enter 
2. Navigate to the registry path shown in Backup_Info.txt 
3. Right-click in the right panel and select New > String Value 
4. Enter the Entry name from Backup_Info.txt 
5. Double-click the new entry and paste the Value 
 
METHOD 2: Command Line Restore 
--------------------------- 
Run Command Prompt as Administrator and use: 
 
reg add "REGISTRY_PATH" /v "ENTRY_NAME" /t REG_SZ /d "VALUE" /f 
 
Example: 
reg add "HKCU\Software\Microsoft\Windows\CurrentVersion\Run" /v "MyApp" /t REG_SZ /d "C:\Program Files\MyApp\app.exe" /f 
 
Replace REGISTRY_PATH, ENTRY_NAME, and VALUE with info from Backup_Info.txt 
 
WARNING: Only restore entries you trust and recognize
